## Notes

- **КРИТИЧНО:** Все компоненты должны использовать shadcn/ui
- Использовать Alert, Card, Dialog, Badge, Progress, Button из shadcn/ui
- Использовать иконки из lucide-react
- Анимации через Framer Motion или CSS transitions
- Подсказки должны быть ненавязчивыми
- Мобильная версия должна быть адаптивной
- Сохранять состояние в БД, не в localStorage (кроме WelcomeBanner)
- Подсказки должны быть контекстными
- Прогресс должен мотивировать пользователя
